# QB-MB-POLICEMDT
MB Police MDT is a premium and professional police tablet for QBCore, built with a clean modern UI, multi-language support, dark and light themes, citizen and vehicle search, reports, BOLOs, warrants, and easy integration for both QB and OX systems.
For more script & updates join the discord. https://discord.gg/NBJtvhyMMW
